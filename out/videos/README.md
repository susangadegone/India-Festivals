# Videos Directory

## How to Add Your India Video

1. **Download a video** from:
   - Pexels: https://www.pexels.com/videos/ (search "India")
   - Pixabay: https://pixabay.com/videos/ (search "India")
   - Unsplash: https://unsplash.com/videos (search "India")

2. **Save it here** as: `india-background.mp4`

3. **That's it!** The landing page will automatically use it.

## Video Requirements

- **Format:** MP4
- **Name:** `india-background.mp4`
- **Size:** Under 10MB recommended
- **Duration:** 10-30 seconds (will loop)
- **Resolution:** 1920x1080 or 1280x720
