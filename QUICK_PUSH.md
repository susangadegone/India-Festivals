## Quick Steps

### 1. Authenticate with GitHub CLI

Run this command and follow the prompts:

```bash
gh auth login
```

Choose:
- **GitHub.com**
- **HTTPS**
- **Login with a web browser**
- Follow the browser prompts

### 2. Push Your Code

Once authenticated:

```bash
git push origin main
```

### 3. Enable GitHub Pages

After pushing:
1. Go to: https://github.com/jaxxat1300/Hindu-Marati-Festivals-App-1/settings/pages
2. Under **Source**, select: **GitHub Actions**
3. Click **Save**

### 4. Watch Deployment

1. Go to: https://github.com/jaxxat1300/Hindu-Marati-Festivals-App-1/actions
2. Watch the workflow deploy your site
3. Your site will be live at:
   ```
   https://jaxxat1300.github.io/Hindu-Marati-Festivals-App-1/
   ```
