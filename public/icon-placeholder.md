# App Icons

To complete the PWA setup, you need to add icon files:

1. `icon-192.png` - 192x192 pixels
2. `icon-512.png` - 512x512 pixels

These icons should be square PNG images. You can:
- Use the calendar emoji 📅 as the base
- Add gradient background (saffron to rose)
- Make them colorful and distinctive

For now, the browser will use a default icon, but custom icons will make the app look more professional when installed.
